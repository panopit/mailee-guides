Integração com Site Através de Formulário

Este artigo irá mostrar como o Mailee.me cria para você um código hmtl pronto para ser integrado ao seu site. Assim os seus contatos podem fornecer seus emails diretamente para sua base de dados do Mailee.me, tudo através do seu Site. Os formulários estão disponíveis para.

  • Cadastro de contato
  • Descadastro de contato

Capítulos

  1. Gerando o Código para Cadastro
    • Gerando Código para Descadastro

      1 Gerando o Código para Cadastro

      Primeiro, vá até a guia configurações da barra de navegação. Selecione Integrar e clique no botão cadastro

      Escolha para qual ou quais listas os contatos devem ser inscritos.

      Lembre-se. Os contatos devem estar inscritos em listas para receber suas mensagens.

      Após ter escolhido as listas, clique no botão Gerar novo Html.

      Antes de copiar o código, localize a terceira e quarta linha. Elas possuem os valores URL_OK e URL_ERROR. Ambos devem ser substituídos.

      URL_OK – Substitua pela página em seu site que aparecerá depois que o contato for inserido com sucesso no Mailee.me.

      URL_ERROR – Substitua pela página em seu site que aparecerá depois que ocorrer algum problema na inserção do contato no Mailee. Será passado o erro por GET pela variável “error”.

      2 Gerando Código para Descadastro

      Não é necessário gerar o código de descadastro de contatos, basta copiar e colar. Não esquecendo de alterar os valores:

      URL_OK – Substitua pela página em seu site que aparecerá depois que o contato for inserido com sucesso no Mailee.me.

      URL_ERROR – Substitua pela página em seu site que aparecerá depois que ocorrer algum problema na inserção do contato no Mailee. Será passado o erro por GET pela variável “error”.
