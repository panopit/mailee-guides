Como realizar upload de template

Esse guia é voltado para pessoas que desejam saber o que é necessário para enviar seus templates para o Mailee.me. Ao final dele você devera saber como:

  • Organizar o conteúdo do zip
  • Fazer upload do template para o Mailee.me
  • Conferir o template enviado

Capítulos

  1. Preparando o arquivo compactado para upload
    • Upload do template
      • Conferindo o seu template

        1 Preparando o arquivo compactado para upload

        Após a criação de seu template, você deverá compactar todos os arquivos em zip para realizar upload para o sistema.

        Seu zip deve, obrigatoriamente, conter apenas:

        • Um arquivo .html (e apenas UM) salvo em UTF-8 (saiba como salvar em UTF-8 em Como criar templates)
        • Arquivos de imagem em formato .gif, .jpg, .jpeg, .png ou .bmp.

        O sistema irá descartar quaisquer arquivos que não preencherem essas especificações.

        2 Upload do template

        1. Clique no botão Upload e um formulário para envio de arquivo irá aparecer
        2. Clique no botão para envio de arquivo*
        3. Selecione seu arquivo em uma lista
        4. Pronto

        Com seu template feito, renomeie o HTML para ‘template.html’ e coloque todas as imagens na mesma pasta. Zipe tudo e envie para o Mailee. O sistema criará uma pasta e descompactará seu conteúdo, ajustando seu HTML ao padrão do Mailee.me.

        Para colocar uma imagem de thumbnail do seu template, faça o seu print screen e coloque o nome ‘template.png’. A imagem deve ir junto no zip. Se você não enviar o thumbnail, não se preocupe, o Mailee tentará gerar um para você.

        Após enviar, você pode verificar como ficou seu layout com notícias de exemplo, ou mesmo fazer um checklist para ver se fez tudo corretamente.

        Importante:
        O arquivo deve se chamar literalmente ‘template.html’ e o thumbnail ‘template.png’. Todas as imagens e arquivos devem estar na mesma pasta.

        * Observação: O nome no botão de envio de arquivo pode ser diferente, dependendo do sistema operacionar e navegador utilizado.
