Cadastro de usuários

Este artigo irá mostrar como é realizada a criação e a edição de usuários.

Capítulos

  1. Tela de usuários
    • Criando usuários
    • Permissões:
  2. Editando usuários

    1 Tela de usuários

    O Mailee possui no menu um item específico para usuários, clicando nele você terá a imagem a seguir

    Nesta tela é possível ver todos os usuários cadastrados no seu subdomínio do Mailee. Se necessário, instalamos um campo de pesquisa para facilitar a localização.

    1.1 Criando usuários

    Clique no botão “Adicionar usuário” para ser direcionado a tela de criação.

    Nessa tela os campos abertos são:

    Nome – Aqui deve ser preenchido o nome completo do usuário.

    Email – O email deve ser preenchido para que o contato receba o lembrete de senha.

    Usário – o nome do usário para acessar a conta do mailee.

    Senha e Confirmação de senha – preencha com sua senha e a confirme.

    1.2 Permissões:

    Quais acessos o usuário terá?

    Relatórios? Acessa os relatórios de envio.

    Mensagens? Permite cadastrar campanhas e enviar mensagens.

    Templates? Permite visualizar e cadastrar templates.

    Contatos? Permite visualizar, cadastrar, importar e exportar contatos.

    Administrador? Acessa conteúdos administrativos, como: pagamento da conta, criação de outros usuários e atualização dos dados do cliente.

    2 Editando usuários

    Para editar um usuário existente, Basta clicar no usuário desejado para ser direcionado a tela de edição. Os campos são os mesmos para adicionar um novo usuário.
