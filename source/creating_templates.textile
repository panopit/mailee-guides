h2. Como criar templates

p. Esse guia é voltado para pessoas que desejam saber como criar seus próprios templates para usar no Mailee.me. Ele cobre funcionalidades que vão do nível básico ao avançado, e ao final dele você deverá saber como:

* Configurar corretamente sua mensagem
* Tornar campos e imagens editáveis
* Criar blocos de repetição
* Saber como inserir links dinâmicos (unsubscribe, weblink, etc) na sua mensagem

endprologue.

h3. Tornando áreas de texto editáveis


p. Imagine que você precisa permitir que o conteúdo de texto, dentro de um elemento <samp>&lt;p&gt;…&lt;/p&gt;</samp>, seja editado, e seu HTML é algo assim:

<html>
  <p>
    Essa informação deverá ser editada
  </p>
</html>

p. Nesse caso, você simplesmente precisará adicionar uma <samp>class="edit"</samp> ao seu elemento <samp>&lt;p&gt;…&lt;/p&gt;</samp>

<html>
  <p class="edit">
    Essa informação poderá ser editada
  </p>
</html>

p. A <samp>class="edit"</samp> não precisa, obrigatoriamente, ser aplicada em elementos de bloco <samp>display:block;</samp>, e pode ser adicionada em elementos <samp>display:inline;</samp>, conforme exemplo abaixo:

<html>
  <p>
    Você também pode definir que <span class="edit">apenas uma pequena porção de texto</span> seja editável.
  </p>
</html>

WARNING. Não use elementos com <samp>class="edit"</samp>, dentro de outros elementos com <samp>class="edit"</samp>.

h4. Possibilitando a troca de uma imagem no template

p. Para que uma imagem pode ser substituida em seu template, basta adicionar a <samp>class="edit"</samp>.

<html>
  <img src="sua_imagem.jpg" class="edit" … />
</html>

h4. Definindo um tamanho para as imagens

p. Se você quiser restringir o tamanho de uma imagem basta imediatamente após a classe 'edit':

<html>
  <img src="sua_imagem.jpg" class="edit 100x100" … />
</html>

p. O formato deve seguir *largura* x *altura*, sendo que, os valores colocados são interpretados por _default_ em pixels, e o tamanho definido como o *máximo* para a imagem no seu layout.

p. Você também poderá usar valores com os seguintes operadores:

|_. *#* | Mantém a proporção da imagem durante o redimensionamento |
|_. *!* | Ignora a proporção da imamge durante o redimensionamento |
|_. *%* | Interpreta os valores como percentual |
|_. *&gt;* | Redimensiona a imagem para o tamanho estabelecido somente se a imagem for *menor* |
|_. *&lt;* | Redimensiona a imagem para o tamanho estabelecido somente se a imagem for *maior* |

p. Para mais informações sobre redimensionamento das imagens usando o ImageMagik, acesse o site "imagemagick.org/Magick++/Geometry.html":http://www.imagemagick.org/Magick%2B%2B/Geometry.html

h3. Blocos de repetição <span class="small" style="color:#a2bf2f;">novidade</span>

p. Blocos de repetição servem para criação de templates onde é necessário repetir o layout, alterando conteúdos.

h4. Definindo um bloco de repetição

p. Para criar um bloco de repetição em seu template basta adicionar uma <samp>&lt;div&gt;</samp> com a <samp>class="repeat"</samp>, um <samp>id=""</samp> que represente o nome do bloco para indexação e um <samp>title=""</samp> que será usado para o botão adicionar na edição de mensagens.

p. Consideremos o seguinte cenário, onde terei um bloco de repetição de notícias:

<html>
  <div>
    <h1> Seu título aqui </h1>
    <img src='sua_imagem.jpg' />
    <p> Conteúdo de seus textos aqui… </p>
  </div>
</html>

p. Para tornar essa <samp>&lt;div&gt;</samp> em um bloco de repetição de notícias, basta adicionar a class, id e title:

<html>
  <div id="news" class="repeat" title="notícias">
    <h1> Seu título aqui </h1>
    <img src='sua_imagem.jpg' />
    <p> Conteúdo de seus textos aqui… </p>
  </div>
</html>

INFO. Dentro dos blocos de repetição não será necessário definir <samp>class="edit"</samp> nos elementos, pois o bloco de repetição, por padrão, já apresenta todos os elementos como editáveis.

p. Recapitulando:

|_. *<samp>class='repeat'</samp>* | torna um elemento <samp>&lt;div&gt;</samp> em um bloco de repetição |
|_. *<samp>id=''</samp>* | define o conteúdo do bloco e aponta para um índice |
|_. *<samp>title=''</samp>* | indica o conteúdo para o botão |

h4. Definindo o índice dos itens no bloco de repetição

p. Você pode incluir em suas mensagens uma lista contendo o índice de seus blocos repetíveis montado a partir dos títulos nos blocos repetíveis.

p. Começamos com o seguinte caso:

<html>
  <div class="index +ID+"></div>
</html>

p. Para começar, é preciso que a <samp>&lt;div&gt;</samp> envoltória dos elementos tenha a <samp>id=""</samp> como <strong>news</strong> assim o Mailee sabe que tudo que estiver dentro dela será repetido para cada notícia. Depois é só colocar um class em cada componente da notícia. As classes de cada campo devem ser: news-title news-image news-content

h3. Links dinâmicos

h4. Descadastramento (unsubscribe)

p. É necessário que o seu email contenha um link para o usuário deixar de receber suas notícias, caso ele queira. Você pode colocar um link para o seu site, ou colocar um link diretamente mara o *Mailee.me*. Se o seu link é assim&hellip;

<html>
  Para deixar de receber a nossa campanha, <a href="#">clique aqui</a>.
</html>

p. Basta adicionar o <samp>class="unsubscribe"</samp>:

<html>
  Para deixar de receber a nossa campanha, <a href="#" class="unsubscribe">clique aqui</a>.
</html>

p. Assim o *Mailee.me* vai trocar o destino do link para o link de  descadastro de cada contato.

INFO. Recomendamos como boa prática sempre incluir ao menos um link para descadastro em suas mensagens.

h4. Visualização na Web (weblink)

p. O link para visualização no browser é indicado para garantir que os usuários que tiverem problemas de visualização com as mensagens no formato HTML, possam visualizar a mensagem no navegador, como uma página de web.

p. Para inserir um *weblink* em sua mensagem, utilize o seguinte código:

<html>
  Para visualizar nossa campanha em seu navegador, <a href="#" class="web-link">clique aqui</a>.
</html>

p. Assim o *Mailee.me* vai trocar o destino desse link para uma url de visualização na web da mensagem.
