h2. Como enviar uma mensagem

Este artigo irá mostrar os passos para se criar e enviar uma nova mensagem.

endprologue.

h3. Criando uma nova mensagem.

Clique em *Mensagens* na barra de navegação. Clique na campanha que deseja utilizar para realizar o envio da mensagem. Caso você queira criar uma campanha nova, clique em _Nova campanha_.

INFO. Para saber como funcionam as campanhas, "clique aqui":what_is_a_campaign.html.

Se você já possui a campanha que deseja e já clicou nela, clique no botão de nova mensagem.

!images/messages/new/message_new_button.png(Nova mensagem)!

h4. Edição de conteúdo

Nesta tela é possível alterar o conteúdo de campos editáveis do template.

h5. Os ícones de edição

!images/messages/new/wysiwyg_icons.png(Wysiwyg)!

*Lápis* - permite você editar diretamente o conteúdo na mensagem.

*HTML* - Abre a edição de HTML para editar a formatação da notícia ou inserir links, por exemplo.

*Disquete* - Salva as alterações realizadas.

*Não permitido* - Abandona a edição atual da notícia, cancelando alterações realizadas.

*Remoção* - Remove uma notícia adicionada ao template.

h4. Conferir

*Número de contatos* - O Mailee.me lhe informa para quantos contatos a mensagem será enviada. Os números podem ser alterados, dependendo da lista e do segmento.

*Título* - O título da mensagem.

*Assunto* - O conteúdo do campo _assunto_ do email.

*Nome do remetente* - O nome de quem envia a mensagem.

*Email do remetente* - O email de quem realiza o envio.

*Email de resposta* - Para onde as mensagens respondidas serão encaminhadas.

*Lista* - Para qual lista será realizado o envio.

*Segmento* - Para qual segmento será realizado o envio.

h5. Preview

Mostra sua mensagem pronta e como seus contatos irão recebe-lá.

h4. Testar

Você pode realizar testes de envio para contatos de suas listas.

h4. Enviar

*Enviar agora* - Inicia imediatamente o envio da mensagem.

*Enviar em* - Envio agendado. Escolha a data e horário do início do envio.

INFO. A qualquer momento do processo de criação de uma nova mensagem é possível salvá-la como *Rascunho* e continuar depois.
